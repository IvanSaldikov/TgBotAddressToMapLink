# Go to Yandex Map Telegram Bot
## Пользователь кидает текстовой адрес в Телеграм-бот, а бот возвращает ссылку на Яндекс.Карты

- Адрес бота: https://t.me/gotoyam_bot
- Имя бота: gotoyam_bot

- Python 3.8
- pyTelegramBotAPI
